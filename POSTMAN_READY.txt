❌ SI ÇA N'OMARCHE PAS
════════════════════════════════════════════════════════════════════════════

BACKEND NE DÉMARRE PAS
  → Vérifiez Node.js installé: "node --version"
  → Vérifiez npm: "npm --version"
  → Supprimez: backend/data/harchive.db
  → Réinstallez: "cd backend && npm install"

POSTMAN ERREUR DE CONNEXION
  → Vérifiez START_BACKEND.bat encore ouvert
  → Vérifiez port 3001 (pas attaqué): "netstat -nao | find 3001"
  → Redémarrez backend si nécessaire

TESTS ÉCHOUENT
  → Vérifiez l'ordre (test 1 avant test 2)
  → Vérifiez l'environnement sélectionné
  → Vérifiez la version Postman (mise à jour?)
  → Essayez: Ctrl+Shift+P → "Clear all cookies"

SI TOUJOURS PAS BON:
  → Consultez POSTMAN_GUIDE.md (troubleshooting section)
  → Consultez TESTING_GUIDE.md (common issues section)
