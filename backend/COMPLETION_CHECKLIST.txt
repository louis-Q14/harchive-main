STEP 1: START BACKEND
───────────────────────────────────────────────────────────────────────────
Open Terminal/PowerShell and run:

  cd backend
  npm run dev

Expected output:
  ✅ Backend initialized
  ✅ Database created
  ✅ Server running on http://localhost:3000

STEP 2: CHOOSE YOUR TESTING METHOD
───────────────────────────────────────────────────────────────────────────
Pick ONE method based on your preference:

A) POSTMAN (EASIEST - Recommended for beginners)
   1. Download Postman: https://www.postman.com/download/
   2. Open Postman
   3. File → Import
   4. Select: HARCHIVE_Backend_Tests.postman_collection.json
   5. File → Import
   6. Select: HARCHIVE_Backend_Environment.postman_environment.json
   7. Click test names on left panel
   8. Click "Send" button
   9. See results in right panel

B) POWERSHELL (BEST FOR WINDOWS)
   1. Stay in backend folder
   2. Run: .\test-backend.ps1
   3. Watch tests run with colored output
   4. See summary at end

C) BASH (BEST FOR LINUX/WSL/MAC)
   1. Open Terminal
   2. cd backend
   3. Run: bash test-backend.sh
   4. Watch tests execute
   5. Colored output shows results

D) MANUAL (FULL CONTROL)
   1. Open Terminal/cmd
   2. Use curl commands from TESTING_GUIDE.md
   3. Use test examples to customize requests
   4. Full transparency of each request

STEP 3: VERIFY RESULTS

All tests should show: PASS ✅

Expected summary:
  ✅ 12/12 tests passed
